# Post-v1.5 improvements — queue

> Status: **All implemented** (v1.6.0, unreleased) · 2026-06-14. A queue of UX / polish / feature work
> agreed after the v1.5.0 device-testing pass; all seven items below are now done, each with its own
> dated plan doc. All kept the no-`INTERNET` invariant.

## Queued (all done)

1. **Share to Stela (receive shared text)** — ✅ **Done (v1.6.0, unreleased).** The app is a
   `text/plain` share target; sharing text opens a new note prefilled with it. Offline, no new
   permission. See [2026-06-14-share-to-stela.md](2026-06-14-share-to-stela.md).

2. **First-run onboarding** — ✅ **Done (v1.6.0, unreleased).** A skippable 3-pane first-launch flow
   (concept → notifications → keep-alive) that explains the concept, requests `POST_NOTIFICATIONS` with
   context, and nudges battery-optimisation exemption. Replayable via Settings → About. See
   [2026-06-14-first-run-onboarding.md](2026-06-14-first-run-onboarding.md).

3. **Editor discard-changes confirm** — ✅ **Done (v1.6.0, unreleased).** Back in the full editor shows a
   "Discard changes?" confirm (Discard / Keep editing) when the note is dirty; Save stays enabled but is
   a no-op that doesn't bump Modified Time when nothing changed. See
   [2026-06-14-editor-discard-confirm.md](2026-06-14-editor-discard-confirm.md).

4. **Scheduled-note list indicator** — ✅ **Done (v1.6.0, unreleased).** A clock icon + relative-time
   label ("Pins in 3h" / "Unpins tomorrow") inline in the row's overline for notes with an auto-pin/
   unpin; main list only. See [2026-06-14-scheduled-note-indicator.md](2026-06-14-scheduled-note-indicator.md).

5. **Material You / dynamic colour** — ✅ **Done (v1.6.0, unreleased).** An optional, off-by-default "Use
   system colours" toggle (Settings → Theme, Android 12+) themes the in-app UI from the wallpaper palette,
   falling back to the brand palette; in-app UI only (widget left as-is). See
   [2026-06-14-material-you.md](2026-06-14-material-you.md).

6. **Haptic feedback on pin toggle** — ✅ **Done (v1.6.0, unreleased).** A light haptic (distinct
   ToggleOn/ToggleOff) when toggling a note's pin on the list row and the shared editor/popup pin button;
   no permission or setting. See [2026-06-14-pin-haptics.md](2026-06-14-pin-haptics.md).

7. **Duplicate note action** — ✅ **Done (v1.6.0, unreleased).** A Duplicate item in the editor's ⋮ menu
   creates an independent copy (fresh id, unpinned, unscheduled, copied title/description/emoji), staying
   on the note with an Undo snackbar. See [2026-06-14-duplicate-note.md](2026-06-14-duplicate-note.md).

## Deferred / not now (recorded for context)

- **Checklist / to-do notes** — out of scope for this project; kept in mind as a possible future
  headline feature.
- **Richer empty state** — current instruction text is considered sufficient.
- **Notification Snooze action** — Snooze stays editor-only for now (this was previously shelved).
- **Swipe actions on list rows** (swipe-to-archive / delete) — good idea, deferred.
